# Clarior

> A Modern CRM Platform

Clarior is a modern Customer Relationship Management (CRM) platform built to help businesses manage customers, leads, tasks, and sales activities through a clean, intuitive, and scalable web application.

## 🚀 Status

🚧 Currently Under Development

## 🎯 Vision

To simplify customer relationship management by providing businesses with a centralized platform that improves productivity, collaboration, and decision-making.

## 🛠️ Planned Tech Stack

- React
- Node.js
- Express.js
- PostgreSQL
- Prisma ORM
- Tailwind CSS
- JWT Authentication
